SAEN = SQL + Angular + Express + Node JS

Sobre el proyecto

El proyecto consiste en una aplicación web que permite gestionar una base de datos de una empresa que trabaja con un stock de productos. La aplicación web se encuentra enfocada en la gestión de productos.
El Backend fue desarrollado como una API REST con Node JS, Express y MySQL. Esta deployado en heroku: https://abm-saen.herokuapp.com/api
El Frontend fue desarrollado con Angular y Bootstrap.

Alta

En esta seccion de puede dar de alta un nuevo producto. El formulario tiene un campo de texto para el nombre del producto, un campo de texto para la descripción del producto,un campo de texto para el precio del producto y un campo de texto para la cantidad de producto. Al dar de alta el producto se muestra un mensaje de confirmación.

Listado

En esta seccion de la aplicación se encuentra una tabla con los productos registrados. En cada item se encuentra un botón para eliminar o editar un producto. Al eliminar un producto se muestra un mensaje de confirmación.
